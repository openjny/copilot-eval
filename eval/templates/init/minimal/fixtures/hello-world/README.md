# fixtures/hello-world/

Files in this directory are copied into `/workspace` inside the container
before the `hello-world` task's prompt runs. This starter task doesn't need
any input files — it only asks Copilot to create one — so this fixture is
intentionally empty (only this README).

Replace it with real project files (a small repo checkout, a config file,
sample data, ...) once you adapt this task to what you're actually
evaluating. See `docs/configuration.md#fixtures` for details, including how
one task can run against multiple fixtures.
